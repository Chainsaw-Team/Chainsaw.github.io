Chainsaw.arithmetic.bitheap

BitHeapCompressor 

case class BitHeapCompressor(arithInfos: Seq[ArithInfo], solver: BitHeapSolver = null) extends UnsignedMerge with Product with Serializable

enhanced multi-operand adder implemented by compress tree using BitHeap model

arithInfos

the sequence of ArithInfo which describe operands information contain weight, width, sign and time

solver

the BitHeapSolver which this multi-operand adder will use, if its null, will search all solver to find a best solver to use
